# CATCH_RUN
(EN)
This game is an arcade game developed using Python, Pygame, and ModernGL (OpenGL).

## Features
- Dynamic Difficulty level (character speed increasing over time).
- High-performance graphics with OpenGL.
- Atmospheric music with sound effects.
- Movement-based reaction (flashing when hitting a bomb).

## Installation
To run the project:
1- 'pip install -r requirements.txt'
2- 'python main.py'
